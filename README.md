# 참새 사냥 게임 만들기 PROJECT!

**Kotlin team project**<br>
*202511070 정유찬* <br>
*202310873 이주현* <br>

진행해야할 기능

    1. 게임 루프 + 스폰 + 이동 + 제거
        1.1 GameView 또는 SurfaceView 생성, 루프(onDraw) 구성
        1.2 deltaTime 계산
        1.3 스폰 타이머 및 최대 개체 수 제한
        1.4 화면 외곽 포인트 랜덤 지정, 반대편 방향으로 속도·각도 설정
        1.5 각 오브젝트의 이동(속도 × deltaTime)
        1.6 화면 벗어나면 자동 제거 (리스트에서 remove)

    2.터치 판정 + 점수 + HUD
        2.1 onTouchEvent 구현
        2.2 히트 성공 시 점수 가산 (+5, -1, -3 등)
        2.3 미스 클릭은 점수 변화 없음 또는 경고음
        2.4 HUD 표시: 점수 / 남은 시간 / 스테이지 이름

    3.스테이지 종료&결과창 저장
        3.1 스테이지 타이머 30초 동작
        3.2 시간 0초 → 게임 정지 상태로 전환
        3.3 기준 점수 달성 시 “성공” 텍스트, 실패 시 “실패”
        3.4 최고 점수 비교 및 SharedPreferences 저장
        3.5 “다시 시작 / 다음 스테이지” 버튼 처리
        3.6 화면 전환 애니메이션 또는 페이드 효과

    4.오브젝트 풀링(좀더 연구 필요함)
        4.1 참새, 까치, 멧새 각각 별도 풀 생성
        4.2 필요 시 obtain() → activeList 등록
        4.3 제거 시 recycle()로 풀에 반환
        4.4 ArrayList 대신 MutableList+풀 캐시로 GC 최소화

    5.무리사냥 + 콤보(선택)
        5.1 필드 내 까치 수 >= 2일 때 무리사냥 모드 플래그
        5.2 참새 랜덤 소멸(점수 획득 방해) 이벤트 주기 설정(대략..)
        5.3 무리사냥 종료 조건(시간/개체 수)
        5.4 콤보 구현 시: 연속 히트 타이머(1.5초), 콤보 배수 계산
        5.5 콤보 HUD 추가, 미스 시 콤보 리셋

    6.스테이지 변경기능
        6.1 StageData 배열로 각 스테이지 구성 (속도/스폰주기/기준점수)
        6.2 스테이지 종료 후 선택된 다음 스테이지 로드
        6.3 배경 이미지 교체 (아침→낮→저녁→밤)
        6.4 변수 초기화 및 재시작 시 타이머 리셋

    6.디버그/최적화/사운드

**🐦 참새 사냥 게임**

🎯 개요

    장르: 클릭형 캐주얼 슈팅

    플랫폼: Android 8.0 이상

    세션 길이: 스테이지당 30초, 총 4스테이지


🕹 플레이 방식

    화면을 탭해 새를 맞혀 점수 획득

    제한 시간 내 최대 점수를 기록


🐦 등장 개체

    개체	점수	특징

    참새	+5	기본 타깃

    까치	−1	2마리 이상 시 ‘무리사냥’ 발동
    
    멧새	−3	빠른 이동, 작은 크기


⚙ 무리사냥

    조건: 화면 내 까치 ≥ 2

    행동: 가장 가까운 참새를 추격(속도 1.3×)

    해제: 3초 경과 또는 까치 < 2

🎯 점수 규칙

    참새 +5, 까치 −1, 멧새 −3

    까치가 참새 사냥 가능함
